### Prerequisites and Usage

1. Clone the repo
   ```sh
   git clone https://github.com/maurbe/ember.git
   ```
3. Install packages
   ```sh
   pip install numpy, matplotlib, tensorflow, tensorflow-gpu
   ```
4. Set the datapaths in `module.py`
5. Train the network
   ```sh
   python train.py
   ```
